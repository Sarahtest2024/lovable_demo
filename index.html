Export to GitHub

Sync your project to GitHub for code handoff and version control:

  • In your Lovable project, click the "Export" or "GitHub" button

  • Choose a repository name (e.g., "checkatrade-plumber-landing")
  • Click "Export to GitHub" to create your repository
  • Share the GitHub repo link with your development team
Why GitHub? This gives your technical team access to the complete codebase for customization, hosting, or integration with existing systems.
